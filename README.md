# CLI Color Calculator

A cli calculator for different color spaces (including sRGB, CMYK, HSL, HSV and CIE Lab (D50))

## Prerequisites

- Go version 1.0 or later 

## Installation

To set up and run this project, follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/Diwice/cli-color-calc
```
2.Navigate to the project directory:
```bash
cd cli-color-calc
```

## Running the Application

Run the application directly :
```bash
go run ./main.go
```
Or build and run it as an app :
```bash
go build -o cli-color-calc

./cli-color-calc
```
